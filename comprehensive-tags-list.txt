## IMPLEMENTATION NOTES

### Tag Naming Conventions
- Use lowercase with hyphens for multi-word tags
- Keep tags concise but descriptive
- Avoid special characters except hyphens
- Use singular form when possible

### Tag Relationships
- Products can have multiple tags from different categories
- Some tags are mutually exclusive (oral vs injectable)
- Consider tag hierarchies for better organization

### SEO Considerations
- Include common search terms as tags
- Use both scientific and common names
- Include brand-specific terminology
- Consider regional spelling variations (UK vs US)
